# React Simple Comps 组件库

## 🎨 简洁优雅的 React UI 组件库

专为现代 Web 应用设计的轻量级 React 组件库，提供丰富实用的 UI 组件，帮助开发者快速构建美观、易用的用户界面。

## 🌟 核心特性

### 💡 简单易用
- 直观的 API 设计
- 详细的文档和示例
- 开箱即用的组件

### ⚡ 轻量高效
- 按需加载支持
- 小体积打包
- 优秀的性能表现

### 🎨 现代设计
- 简洁美观的视觉风格
- 响应式设计支持
- 主题定制能力

### 🛠️ 技术优势
- 完整的 TypeScript 支持
- 严格的类型检查
- 现代浏览器兼容


## 🚀 快速开始

```bash
npm install react-simple-comps
```
```js
import { Button } from 'react-simple-comps';

function App() {
  return (
    <Button type="primary">
      Hello React Simple Comps!
    </Button>
  );
}
```
